# TaskFlow

![Integración continua](https://github.com/SarAbigail/taskflow/actions/workflows/ci.yml/badge.svg)

Sistema web para la gestión de tareas, desarrollado como proyecto del curso Gestión de Plataformas de Soporte al Desarrollo.

## Tecnologías

- Java 21
- Spring Boot
- Thymeleaf
- Bootstrap
- PostgreSQL
- Docker
- Maven
- Git
- GitHub Actions

## Funcionalidades

- Registro de tareas
- Edición y eliminación
- Clasificación por prioridad
- Gestión por estado
- Fecha de vencimiento
- Búsqueda por título
- Filtros por estado y prioridad
- Dashboard de tareas
